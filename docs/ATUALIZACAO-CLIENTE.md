# Star Fire - Atualização Offline (sem perder dados)

## Como funciona
- O aplicativo instala em `C:\Program Files\Star Fire`.
- Os dados do cliente ficam em `%LocalAppData%\StarFire\data`.
- Ao atualizar versão, os dados permanecem intactos.

## Passo a passo para o cliente (leigo)
1. Feche o Star Fire.
2. Dê duplo clique no arquivo `StarFire-Setup-<versão>.exe`.
3. Clique em `Avançar` até finalizar.
4. Abra o Star Fire normalmente.

Pronto. Os clientes/cobranças/configurações continuam no sistema.

## Recuperação rápida (se necessário)
- Backups automáticos/manuais ficam em `%LocalAppData%\StarFire\data\backups`.

## Observação importante
- Não apagar manualmente a pasta `%LocalAppData%\StarFire\data`.
- Essa pasta contém o banco e as configurações do cliente.
